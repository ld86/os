CPU
===

Файлы:
* sample
  Слепок памяти, который будет загружен.
* config
  Настройка маскировки проводков.
  Должен быть в формате:

  PRZNK=[0|1]

  Например:
  
  ZAM1=0
  <br/>
  ZAM2=1

  Доступные проводки:
    OP
    PEREH
    PUSK
    ZAPP
    ZAM1
    ZAM2
    VZAP1
    VIB
    CHIST
